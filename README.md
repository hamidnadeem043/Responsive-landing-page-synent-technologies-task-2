# Smart Watch Landing Page ⌚

A modern and responsive **Smart Watch product landing page** built using **HTML** and **CSS**.

This project showcases a product landing page for **Smart Watch Pro** with a clean UI, responsive layout, and feature highlights.

## 🚀 Features

* Hero section with product title and description
* Call-to-action **Buy Now** button
* Feature cards section
* Hover effects on cards and button
* Fully responsive design for:

  * Desktop
  * Tablet
  * Mobile
* Clean footer section



## 🛠 Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Media Queries

## 📱 Responsive Design

This landing page is responsive across different screen sizes:

* **Desktop:** 4 feature cards in one row
* **Tablet:** 2 feature cards in one row
* **Mobile:** 1 feature card per row
